Welcome!

FMM is a simplified version of Windows Explorer. With it, you can perform file system operations faster. Use the command number and its arguments for the operation.

Introducing FMM

Revolutionizing File Management for Windows, FMM offers robust encryption algorithms that transform your plaintext data into indecipherable ciphertext, ensuring your information remains secure. With support for various encryption methods, including AES, RSA, and Blowfish, you can encrypt files and messages with confidence.

Decode with Ease

Decrypting data is seamless with FMM's intuitive functionality. Whether retrieving encrypted files or decrypting messages, the process is hassle-free. Key management and passphrase protection ensure only authorized users access sensitive information.

Seamless File Transfers

Gone are the days of cumbersome file transfers. FMM simplifies moving files between folders, drives, or cloud storage platforms, ensuring efficient transfers.

Secure File Handling

Security is paramount at FMM. Your files are handled with care, with robust encryption protocols safeguarding your privacy.

Note: If installed in a folder on drive C (Windows), always run as an administrator to avoid closure on function calls.

This program is written in Golang.
